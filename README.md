# 📝 SongSong-Notion

## 프로젝트 소개

- 계층형 구조로 게시글을 작성하고 관리할 수 있는 게시판 프로젝트입니다.

## 배포

- [API지원중단으로 정상적으로 작동하지 않습니다.](https://thddlmy.github.io/songsong-notion/)

## 기술 스택

- HTML5
- CSS3
- JavaScript

## 실행 방법

- `yarn start` 명령어로 실행합니다.
